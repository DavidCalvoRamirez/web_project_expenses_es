# TripleTen web_project_expenses

# Descripcion

Esta aplicación permite gestionar y analizar gastos personales de forma sencilla. A través de un conjunto de funciones en JavaScript, el sistema calcula el total de gastos, el promedio, el balance disponible y analiza los gastos por categoría.

# Tecnologías utilizadas

-JavaScript (Vanilla JS)

-Lógica basada en arrays y funciones

# Posibles mejoras

-Persistencia de datos (LocalStorage o base de datos)

-Gráficas estadísticas

-Sistema de autenticación de usuarios
